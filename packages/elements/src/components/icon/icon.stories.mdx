# Icon

The basic icon component relies on adding an `icon` prop that maps to the name of a SVG file. See the tables below for the icons available.

For non-React users, we export all the icons from `@reapit/elements/assets` where they are grouped in folders `animated-icons`, `infographic-icons`, `menu-icons` and `system-icons` eg `import addSystem from "@reapit/elements/assets/system-icons/addSystem.svg"`. You can then wrap your SVG in the appropriate classes you see in the rendered markup with each story in these docs.

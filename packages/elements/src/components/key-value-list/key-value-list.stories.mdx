# KeyValueList

Really a collecation of previously defined components, but exported as a React shorthand for a best practice pattern to display key value pairs as record, for example a contact or property.

The component supports any `ReactNode` for content or icon props so you can use custom elements like links or external icons as required.

The basic usage variant will display the pairs in a list and can be used in a `Drawer` or narrow container.

Adding the `textElipsis` prop will add a text overflow ellipsis to the value, trimming the text to a single line.

<ArgsTable of={KeyValueList} />

```ts
export interface KeyValueItem {
  key: string
  value: ReactNode
  iconName?: IconNames
  icon?: ReactNode
  intent?: Intent
  colSize?: 'half' | 'full'
  textEllipsis?: boolean
}
```

## Usage Grid

The grid variant where you are displaying in a full page view or main container - to achieve this, pass the `hasGrid` prop. For each item, you can specify a `colSize` prop of either 'half' or 'full', depending on the width of your content.

## With Intent

Different intents are supported, to enable buttons to be used in different contexts. Typcially you will always choose an intent for your button. The intent prop maps directly to the intent helper classes for non React users.

<Canvas>
  <Story name="Intent: Primary">

  </Story>

    <Button intent="secondary">Secondary</Button>
  </Story>

## Other attributes

All other standard HTML attributes for `<button>` are supported and are passed through to the React component, e.g. type and onClick

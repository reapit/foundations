## With Intent

Different intents are supported, to enable buttons to be used in different contexts. Typcially you will always choose an intent for your button. The intent prop maps directly to the intent helper classes for non React users.

Whereas most components support the full range of intents, `Buttons` only support `primary`, `danger` and `default`, the latter being the default.

<Canvas>
  <Story
    name="Intent"
    args={{
      intent: 'primary',
      children: 'Button Text',
    }}
  />
</Canvas>

## Button Size

If you wish to size up or down a button, you can use the `buttonSize` prop.

By default all of our buttons are sized to `small` in mobile and `medium` in desktop so this option is solely if you wish to force the size.

<Canvas>
  <Story name="Button Size">

      <Button buttonSize="medium">Medium Button</Button>
      <Button buttonSize="large">Large Button</Button>
    </ButtonGroup>
  </Story>
</Canvas>

## Button Icons

If you wish to add a system icon from the `IconList`, you can do this by passing the `buttonIcon` prop.

You have the option to specify a position of `left`, `right` or `only` as a postion for your icon.

<Canvas>
  <Story name="Button Icons">
    <ButtonGroup>
      <Button
        intent="primary"
        buttonIcon={{
          icon: 'addSystem',
          position: 'left',
        }}
      >
        Left Icon Primary
      </Button>
      <Button
        buttonIcon={{
          icon: 'addSystem',
          position: 'right',
        }}
      >
        Right Icon Default
      </Button>
      <Button
        buttonIcon={{
          icon: 'addSystem',
          position: 'only',
        }}
      >
        Only Icon
      </Button>
    </ButtonGroup>
  </Story>
</Canvas>

<RenderHtmlMarkup component="Button" story={['Button Icons']} />

# Tile

A layout component for rendering generic content. It is typically used with a `hasGreyBackground` container (see layout) and a `Grid` to space tiles with desired effect.

## Padding Variants

For smaller tiles with more to a grid row, you may wish to consider setting the `paddingSize` prop to `small` or even `none` for self sizing components.

### Modifiers

We support a number of modifier to override the default typography, specifically bold, italic, grey coloured and centered text. Also, removing the bottom margin when you want to manually group your text items and adding a 60px bottom margin for denoting the end of a page section.

See the table below for details of these modifiers;

<ArgsTable of={Title} />

In all cases they map to css classes you can see in the rendered markup. Below you can see each of the modiifiers applied to a title and a body text component. They are available also for small text and subtitle.

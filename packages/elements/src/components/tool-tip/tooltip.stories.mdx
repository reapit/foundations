# Tool Tip

## Basic Usage

<ArgsTable of={ToolTip} />

A tool tip is used for hovering over text and displaying additional info. Below is an example of what the tool tip looks like.

<Canvas>
  <Story name="Basic Usage">
    <ToolTip tip="Some Data">Hover here</ToolTip>
  </Story>
</Canvas>

<RenderHtmlMarkup component="ToolTip" story="Basic Usage" />

## Default Active

<ArgsTable of={ToolTip} />

If you want the initial state of the tooltip to be active, you can pass the `defaultActive` prop.

<Canvas>
  <Story name="Default Active">
    <ToolTip tip="Some Data" defaultActive>
      Hover here
    </ToolTip>
  </Story>
</Canvas>

<RenderHtmlMarkup component="ToolTip" story="Default Active" />

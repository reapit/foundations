# InputAddOn

The `InputAddOn` component is intended to append text to input components.

It makes sense largely when used in conjunction with an `InputGroup` component - refer to `InputGroup` for complete input example.

<ArgsTable component={InputAddOn} />

## Basic Usage

On a basic level, it renders a text block at the end of an input.

  <Story name="Basic Usage">

## With Icon

It may also be useful to render an icon after your input - for example to render an error indicator.

## With Intent

If you want add an intent from the intent list, this will add color, for example on a form validation.

Can be combined with an icon to color the icon.

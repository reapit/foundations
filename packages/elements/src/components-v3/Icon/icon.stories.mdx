# Additional className

Set the `className` prop to add additional custom CSS classes

<Canvas>
  <Story name="Additional className">
    <Icon className="my-custom-className" icon="add" />
  </Story>

# Other attributes

All other standard HTML attributes for `<span>` are supported and are passed through to the React component, e.g. onClick
